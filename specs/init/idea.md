Nombre de la app: SportyKids.

Meta: App de noticias deportivas para niños (8–12 años), con feeds personalizados de prensa a elegir, Reels, Quiz y control parental.

Onboarding:

Edad (6–8, 9–11, 12–14, 14-16) o con otros rangos más amplios.

Deportes favoritos.

Equipo favorito.

Medios de prensa a incluir.

Formatos (solo titulares, con imágenes, con vídeos/Reels).

Funcionalidades MVP v1:

Home (feed configurado).

Reels (vídeos cortos).

Quiz (3–5 preguntas por noticia).

Sección “Mi equipo”.

Control parental (tiempo de uso, bloqueo de formatos).

Tecnología:

Front: React Native 

Backend: Python FastAPI + Postgres / CockroachDB 

Autenticación: elección tuya
